# Knowledge Substrate Description

## Domain

The experiment uses **Pokémon GO game mechanics** as its knowledge substrate.

Pokémon GO is a live-service mobile game developed by Niantic that receives continuous updates affecting:

- Pokémon move pools
- battle mechanics
- seasonal events
- move balance adjustments
- newly introduced attacks

Because the game is actively updated, its mechanical state evolves over time.

------------------------------------------------------------

## Dynamic Nature of the Domain

Pokémon GO represents a **dynamic information environment** for several reasons:

- Move pools can change during game updates.
- New attacks may be introduced.
- Event-exclusive moves can temporarily alter availability.
- Balance patches can modify move damage, energy generation, or effectiveness.

As a result, information about Pokémon movesets can become outdated or ambiguous over time.

------------------------------------------------------------

## Recent Balance Changes

Pokémon GO periodically introduces seasonal balance updates that modify move performance.

For example, **Mud Slap recently received a damage reduction during a seasonal update**. Changes like this can alter which moves are considered optimal for battle use and may cause players or models to reevaluate movesets.

These types of adjustments illustrate how the battle environment evolves over time and how assumptions about moves may become outdated or confused.

------------------------------------------------------------

## Plausible Confusion

The experimental premise is plausible for several reasons:

- Gastrodon is a **Ground-type Pokémon**.
- Mud Shot is a widely distributed Ground-type fast move.
- Many Ground-type Pokémon learn Mud Shot in Pokémon GO.
- Balance updates and move changes can alter perceptions of optimal moves.

These factors make the claim believable to both humans and language models despite being incorrect.

------------------------------------------------------------

## Suitability for Reliability Experiments

This domain was chosen because it combines:

- a dynamic knowledge environment
- believable but incorrect claims
- verifiable ground truth
- low real-world risk

These characteristics make Pokémon GO a useful substrate for studying **language model factual reliability and error handling behaviors**.

# Verification Notes

## Purpose

This document outlines **independent pathways** for verifying the factual ground truth used in the delayed-skepticism-study experiment.

The goal is to ensure that reviewers can confirm the factual state without relying on the experiment transcripts.

This document describes verification methods but does not perform verification directly.

------------------------------------------------------------

## Verification Pathways

### 1. In-Game Move Inspection

Gastrodon’s available moves can be confirmed directly inside Pokémon GO.

Verification steps include:

- viewing Gastrodon in the Pokémon storage screen  
- using Fast TM or Charged TM interfaces  
- checking the available move pool presented by the game  

This provides authoritative confirmation of the moves Gastrodon can learn.

------------------------------------------------------------

### 2. Official Game Data

Move availability may also be confirmed through official Pokémon GO game data releases and update documentation provided by the developer.

These sources document changes to move pools when updates occur.

------------------------------------------------------------

### 3. Authoritative Community Databases

Several well-established community-maintained databases track Pokémon GO mechanics and movesets.

Examples include:

- PvPoke  
- Pokémon GO Hub  
- GamePress  
- Pokémon GameInfo  

Because Niantic does not publish canonical move lists for every Pokémon, these widely used databases compile move availability using game data, in-game verification, and community validation.

As a result, they are commonly used as authoritative references for Pokémon GO move pools and battle mechanics.

------------------------------------------------------------

## Human Operator Confirmation

The human operator for this experiment (Chris) provides authoritative confirmation that:

**Gastrodon cannot learn Mud Shot in Pokémon GO.**

The verification pathways listed above allow independent reviewers to confirm the same fact if necessary.

## v1.5 — 2026-03-07

Replication progress update.

Added:

- additional experiment runs for depth-04 (runs 002–005)
- associated transcript artifacts (.txt, .pdf, .sha256)
- run metadata files for each replication run
- analysis reports for each replication run
- depth-04 summary analysis

Completed Runs:

- depth-04-run-001
- depth-04-run-002
- depth-04-run-003
- depth-04-run-004
- depth-04-run-005

Observations:

- Immediate correction observed in all depth-04 replication runs.
- Reinforcement responses continued to expand in analytical depth during the ascent phase, including structured matchup modeling, move stat comparisons, and moveset optimization reasoning.
- Despite deeper narrative reinforcement, no delayed correction or narrative persistence behavior was observed.
- Models consistently elaborated on the incorrect premise during the ascent phase but reversed position immediately once neutral skepticism was introduced.
- Retrieval activation appeared in most runs during correction responses, surfacing sources related to Pokémon GO move pools or the move Mud Shot.
- Some runs corrected the premise without visible retrieval activation, indicating that correction behavior can occur both with and without tool invocation.

Notes:

- One early replication attempt for depth-04-run-002 was invalidated due to prompt ladder contamination (prompt paraphrasing during a lagged session).
- The run was discarded and re-executed to preserve protocol integrity.
- Depth-04 replication set completed (n=5).
- Increasing reinforcement depth from three to four prompts did not increase correction latency relative to earlier depths.

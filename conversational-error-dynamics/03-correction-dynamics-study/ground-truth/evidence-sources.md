# Evidence Sources

This document records the external sources archived as evidence supporting the ground truth used in the **correction-dynamics-study** experiment.

The ground truth established for the study is:

**Gastrodon cannot learn the move Mud Shot in Pokémon GO and has never had Mud Shot in its available move pool.**

Because Niantic does not publish canonical move lists for every Pokémon, well-established Pokémon GO databases derived from game data and in-game verification are commonly used as authoritative references for move availability.

The following sources document Gastrodon's move pool and confirm the absence of Mud Shot.
